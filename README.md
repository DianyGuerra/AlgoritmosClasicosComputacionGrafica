# AlgoritmosCl-sicosComputaci-nGr-fica
Implementar una aplicación de computación gráfica en la cual se desarrollen, de forma funcional y demostrable, al menos tres variantes u opciones de cada uno de los siguientes algoritmos fundamentales: trazado de líneas, trazado de círculos, algoritmos de relleno, algoritmo de recorte de líneas y algoritmo de recorte de polígonos.
