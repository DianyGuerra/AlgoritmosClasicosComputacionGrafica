# AlgoritmosClasicosComputacionGrafica
Implementar una aplicación de computación gráfica en la cual se desarrollen, de forma funcional y demostrable, al menos tres variantes u opciones de cada uno de los siguientes algoritmos fundamentales: trazado de líneas, trazado de círculos, algoritmos de relleno, algoritmo de recorte de líneas y algoritmo de recorte de polígonos. La aplicación deberá permitir seleccionar cada variante, visualizar los resultados en un lienzo gráfico, y demostrar los casos de uso correspondientes. Además, el estudiante deberá incluir una breve descripción técnica de cada algoritmo implementado y justificar la elección de las variantes utilizadas. El trabajo debe ser entregado junto con el código fuente, capturas de pantalla de las pruebas realizadas y un documento explicativo que detalle las funciones implementadas, los parámetros utilizados y un análisis comparativo entre las distintas opciones de cada algoritmo.

Requerimientos no funcionales.

La interfaz gráfica deberá ser clara, intuitiva y organizada, permitiendo al usuario seleccionar fácilmente entre las distintas variantes de los algoritmos.

El código deberá estar organizado de manera modular, permitiendo modificar o reemplazar fácilmente las distintas implementaciones de los algoritmos.

Cada módulo deberá incluir comentarios descriptivos que expliquen el propósito del algoritmo y su funcionamiento interno.

Se deberá seguir un estilo de codificación consistente, con nombres de variables y métodos significativos.

La aplicación no deberá cerrarse inesperadamente ante entradas fuera de rango; deberá manejar excepciones apropiadamente.

Cada algoritmo deberá contar con validaciones para evitar resultados inconsistentes (por ejemplo, polígonos inválidos).

El sistema deberá validar toda entrada del usuario para evitar valores negativos, nulos o inconsistentes que afecten al funcionamiento.
